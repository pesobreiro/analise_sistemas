Pesquisar Empresa

UMLUseCase
Untitled :: Model1 :: Pesquisar Empresa

Description

  1. O utilizar acede à página para pesquisar, seleciona o tipo de pesquisa: nome, tipo empresa, localidade ou região.

  2. Ao inserir 3 letras o sistema apresenta as correspondências, onde o utilizador pode selecionar uma empresa ou escrever as letras restantes até encontrar a empresa

  3. Ao selecionar o nome da empresa apresenta os detalhes

Pós condição : Uma empresa selecionada
